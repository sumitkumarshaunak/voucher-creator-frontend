# Voucher Creator Frontend

React/Vite frontend for uploading voucher documents, previewing the selected file, parsing it through the backend, and editing generated vouchers.

## Setup

```bash
cd /Users/amanjain/Desktop/voucher/voucher-frontend
npm install
```

Optional `.env`:

```env
VITE_API_BASE_URL=http://127.0.0.1:8001
```

## Run

```bash
npm run dev -- --host 127.0.0.1
```

Open the local URL printed by Vite.

## Build

```bash
npm run build
```
